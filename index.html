Who I am

Hello! My name is Richard Shipe and I'm a front-end developer living in Northern Virginia. I was never fully aware of web developments' boundless capacity for creativity and critical thinking until a friend introduced me to coding in 2014. I dabbled in introductory programming MOOCs for a little while and enjoyed making fun applications such as this memory card game. Around mid to late 2015, I realized that web development is something I deeply enjoyed more than my own college major (economics). It allows me to express myself, forces me to be patient with problem solving, and pushes me to meet other developers. With this realization in mind I decided to pursue web development professionally and make a living doing what I love.


When I'm not focusing on programming I'm usually reading about economics, cooking, or playing video games with my friends.

Some Recent Projects

Google Maps Project

Tools used: HTML, CSS, Javascript, jQuery, KnockoutJS, Google Maps API, Wikipedia API

Frogger Clone

Cross the river and avoid the man-eating beatles!

Etch A Sketch

First choose the sketching mode and canvas dimensions. Then hover your mouse over the canvas to doodle.


Educational purpose: Practice event driven jQuery and Javascript. (A side benefit was getting a tad better at using CSS's display property to position elements.)

I was given a dummy site and some conditions to test. Using the Jasmine testing suite, I wrote tests that would look for errors in the provided website. The main difficulty of this project was ensuring that tests were strong and very rigorous.

Tools used: Javascript, Jasmine testing suite

Web Optimization Project
